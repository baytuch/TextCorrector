Это приложение выполняет поиска заданных слов и выделяет их жирным шрифтом.

Для того, чтобы воспользоваться богатыми возможностями программы необходимо скопировать исходный текст в верхнее поле. В нижнем поле нужно указать ключевые слова (вводить с новой строки).

Нажимаем «Обработать» и получаем код для вставки на сайт. Далее ищем в визуальном редакторе кнопочку «HTML» и переносим результат работы проги в окошко.

Код еще довольно сырой. Набил за два часа. Потому косяков хватает. Если будет востребовано, доделаю.
